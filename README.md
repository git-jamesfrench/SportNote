![Sportnote](https://assets.jamesfrench.fr/sportnote/github_header.png)

Sportnote is an app made to make tracking exercises and training a lot easier. In weightlifting, you can write down the weight you lift. In running, you can input all the short runs you make in one session. And with this data, you can see your progression with statistics and see your previous exercises.
